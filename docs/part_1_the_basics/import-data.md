Import Data
-----------

Next, let’s import some data so we can start using the tool. We will be using transcripts from interviews and focus groups run for a US data center study. As mentioned before, NVivo helps you get organized, so before we import in some data, let’s create some folders to organize our data.

1. From the left menu, **under Data, hold down the** **Control key** and **click on Files** and **select New Folder**... from the context menu. Let’s call it “Interviews” and **click on Done**.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_010.png' | relative_url }}" alt='Under Data, files is selected. From there, New Folder is selected' title='' width='1354' height='1060' /><img src="{{ '/assets/images/NVivo15_Mac_intro_011.png' | relative_url }}" alt='The name of the new folder is Interviews' title='' width='1502' height='1120' />

2. **Create one more folder called “Focus Groups”**.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_012.png' | relative_url }}" alt='Two folders have been created, Interviews and Focus Groups' title='' width='998' height='1128' />

3. Next, **go to the Import menu** from within the program and select **Files**.

    **Note:** If you do not have Files selected in the Data tab on the left, all options under Import will be greyed out.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_013.png' | relative_url }}" alt='The import tab has been selected within NVivo, and the files tab has been selected from there.' title='' width='1806' height='1190' />

4. Here you can see some various types of files that NVivo can import. To start, let’s import in our interview transcripts. **Make sure that the Interviews folder is selected on the left**. **Go the Import menu, and from the Files drop\-down menu, select Documents...**

    <img src="{{ '/assets/images/NVivo15_Mac_intro_014.png' | relative_url }}" alt='Under Import, Documents is selected. Make sure that the Interview file is selected as well. ' title='' width='1616' height='1200' />

5. **Browse to the folder called “DCEE Focus Groups and Interviews,”** which has all the study files we will look at today.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_015.png' | relative_url }}" alt='Import Documents window at the workshop folder, with the DCEE Focus Groups and Interviews folder highlighted.' title='' width='1758' height='1176' />

6. **Highlight only the files starting with “Data Center Interview”** **in the title** in our folder by **selecting the first one, holding down the Shift key**, and then **selecting the last one**. You should have 7 files highlighted. Then **select Import**. This may take a moment to complete.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_016.png' | relative_url }}" alt='Import Documents window at the DCEE Focus Groups and Interviews folder, all Data Center Interview files highlighted and selected. Red box around the Import button.' title='' width='1758' height='1176' />

7. Let’s do something similar to import in the focus group information. **Select the Focus Groups** folder on the left. Then **go to the Import menu, and from the Files drop\-down menu, select Documents...**

    <img src="{{ '/assets/images/NVivo15_Mac_intro_017.png' | relative_url }}" alt='Under Data, Focus Groups are selected. Under the Import tab, Files and then Documents is selected. ' title='' width='1584' height='1158' />

8. You should still be in the same folder as last time. This time **highlight the files starting with “NYC”** **in the title**. You should have 8 files and a mix of PDFs and Word Documents. Then **select Import** to import those focus groups files into the Focus Groups folder.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_018.png' | relative_url }}" alt='Import Documents window at the DCEE Focus Groups and Interviews folder, all NYC Data Center files highlighted and selected. Red box around the Import button.' title='' width='1758' height='1176' />

9. These files are now all copied into your project file. You can see from the Import menu that there are many different file types that can be imported. For example, you can connect to reference managers to pull in your data (found under the bibliography tab), or from the Files option, you can import in audio files, images, or videos. We won’t be covering them all in this tutorial, but be sure to explore your options later.

    **Note:** Files, Classifications, Bibliography, and Notes all have drop down menus with additional import types.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_019.png' | relative_url }}" alt='Red box outlining the options under the Data menu.' title='' width='1746' height='230' />

    Note: If you’re interested in learning more about importing other common data sources into NVivo, check out our [tutorial on importing webpages as PDFs and survey data](https://mdlutoronto.github.io/nvivo-14-15-mac-importing-alternative-sources/)!
