Explore the Interface
---------------------

1. Let’s briefly look around before we get started. First off, the left side menu helps you navigate the various items of your project, such as your files, your codes, notes, queries, etc. We will talk about these sections as we use the tool.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_005.png' | relative_url }}" alt='NVivo new project homepage ' title='' width='2878' height='1670' />

2. If you want to save your project manually (which you should do periodically), or create or open a new project, you can use the File menu from the menu bar at the top of the screen.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_006.png' | relative_url }}" alt='Under file, Save is selected ' title='' width='2878' height='1670' /><img src="{{ '/assets/images/NVivo15_Mac_intro_007.png' | relative_url }}" alt='Under file, Save is selected ' title='' width='2880' height='1668' />

 

3. If you select **NVivo from the** **menu bar**, and then **Licensing**, you will see an option to **Replace License**, where you can update your license key, when you receive a new one.

    <img src="{{ '/assets/images/NVivo15_Mac_intro_008.png' | relative_url }}" alt='In the top menu bar, NVivo is selected' title='' width='1746' height='1118' /><img src="{{ '/assets/images/NVivo15_Mac_intro_009.png' | relative_url }}" alt='Under the NVivo tab, Licensing and Replace license are selected ' title='' width='1518' height='1106' />
